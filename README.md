# RUUMARKER

図：キズのマーク

引越しの際に新居に既にキズがあった場合に、不動産屋に報告するための書類を作成するためのサービスです。
キズの位置とその写真が載っている検査表を作成してPDFでダウンロードできます。

## 利用方法
１．
間取りをアップロードして気になる箇所をチェックしましょう

図

２．
部屋を調べて気になるところの写真を撮りましょう

図

３．
間取り図に、写真を撮った箇所の印をつけましょう。

図

４．
気になるところの説明を詳しく記載しましょう

図

５．
PDFでダウンロードしましょう。

## インストール

```
$ bin/setup
$ rails server
```
